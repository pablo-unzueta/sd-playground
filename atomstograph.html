OneHotAtomSymbols

 OneHotAtomSymbols (symbols:list)

Generate one-hot features for a list of atom symbols.

Type Details
symbols list atom symbols
Returns ndarray [M, N] where M is the number of atoms, and N is the number of unique atom symbols

ScoreDynamicsDataset

 ScoreDynamicsDataset (root, cutoff=5.0, max_num_neighbors=50,
                       transform=None, pre_transform=None,
                       pre_filter=None)

Dataset base class for creating graph datasets which easily fit into CPU memory. See here <https://pytorch-geometric.readthedocs.io/en/latest/tutorial/ create_dataset.html#creating-in-memory-datasets>__ for the accompanying tutorial.

Args: root (str, optional): Root directory where the dataset should be saved. (optional: :obj:None) transform (callable, optional): A function/transform that takes in a :class:~torch_geometric.data.Data or :class:~torch_geometric.data.HeteroData object and returns a transformed version. The data object will be transformed before every access. (default: :obj:None) pre_transform (callable, optional): A function/transform that takes in a :class:~torch_geometric.data.Data or :class:~torch_geometric.data.HeteroData object and returns a transformed version. The data object will be transformed before being saved to disk. (default: :obj:None) pre_filter (callable, optional): A function that takes in a :class:~torch_geometric.data.Data or :class:~torch_geometric.data.HeteroData object and returns a boolean value, indicating whether the data object should be included in the final dataset. (default: :obj:None) log (bool, optional): Whether to print any console output while downloading and processing the dataset. (default: :obj:True)

Type Default Details
root folder where the dataset should be stored. Splits into raw_dir and processed_dir
cutoff float 5.0 cutoff radius for the neighbor list
max_num_neighbors int 50 maximum number of neighbors for each atom
transform NoneType None dynamically transforms the data object before accessing (e.g. data augmentation)
pre_transform NoneType None applies transformation before saving to disk (e.g. best used for heavy precomputations)
pre_filter NoneType None filter out data objects before saving to disk

Do Not Skip: Creating InMemory dataset.

In order to create an ScoreDynamicsDataset, one needs to set up a data directory with two subdirectores: raw and processed. Inside raw, place the ase .extxyz file called “data.extxyz”. Then pass that data directory to the ScoreDynamicsDataset. Torch-Geometric Handles the rest

dataset = ScoreDynamicsDataset(root="../data_path")
